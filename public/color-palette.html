7 — Rules: Do & Don't

✓ Do

  • Use red (#C8102E) only for CTA buttons
  • Use gold on dark backgrounds (#F0C842)
  • Use gold on light backgrounds (#D4A017)
  • Force color: white via inline style on headings over dark bg
  • Use btn-outline-white for ghost buttons on dark bg
  • Alternate sections: white → bg-alt → white
  • Cards always on white bg with border #E2E8F0

✗ Don't

  • Don't use red for text, icons, or decorative elements
  • Don't use text-yellow-400 — use gold CSS variables instead
  • Don't use btn-outline on dark backgrounds (navy text invisible)
  • Don't rely on text-white Tailwind class for headings — the global h1 rule overrides it
  • Don't hardcode hex values — use CSS variables
  • Don't use gold as a background colour
